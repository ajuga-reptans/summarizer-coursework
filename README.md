## Описание проекта

Сервис автоматически генерирует краткое содержание (саммари) веб-статей на русском языке. Система принимает URL страницы, извлекает основной текст, обрабатывает его с помощью предобученной трансформерной модели и возвращает связное резюме объёмом ~20–30% от исходного текста.

### Решаемые задачи

- **Абстрактная суммаризация** (abstractive summarization) — модель не просто копирует предложения, а переформулирует содержание
- **Обработка длинных текстов** — автоматическое разбиение на фрагменты (chunking) для статей >2000 символов
- **Извлечение контента** — умный парсинг HTML с удалением рекламы, меню и служебных элементов
- **Контроль качества** — дедупликация предложений, подавление повторов, настройка параметров генерации

### Практическое применение

- Быстрое ознакомление с новостными статьями
- Мониторинг информационного поля
- Подготовка дайджестов
- Интеграция в аналитические системы

### Troubleshooting

## Ошибка: «Не удалось извлечь текст по URL»
    Причины:
        Сайт блокирует запросы (403 Forbidden)
        Неправильный URL
        Tайм-аут соединения
    Решение:
        Проверьте доступность URL в браузере
        Попробуйте другой источник (Lenta.ru, RIA.ru работают стабильно)
    
## Ошибка: «Read timed out»
    Причина: Frontend ждёт ответа только 60 секунд, а генерация занимает больше времени.
    Решение:
        Для длинных статей (>5000 символов) просто подождите 2–3 минуты
        Не обновляйте страницу и не нажимайте кнопку повторно
